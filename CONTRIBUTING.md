# Contributing to Overwatch Tactical Analysis and Visualization

Thank you for your interest in contributing to the Overwatch Tactical Analysis and Visualization project! We welcome contributions from everyone. Please follow the guidelines below to ensure a smooth and effective contribution process.

## How to Contribute

### 1. Fork the Repository
   - Click on the **Fork** button at the top right of the repository page to create your own copy of the project.

### 2. Clone Your Fork
   - Use the following command to clone your forked repository:
     
     ```bash
     git clone https://github.com/YOUR-USERNAME/overwatch-tac.git
     ```

### 3. Create a New Branch
   - Create a new branch for your feature or bug fix:
     
     ```bash
     git checkout -b feature/your-feature-name
     ```

### 4. Make Your Changes
   - Make the necessary changes to implement the feature or fix the bug. Remember to follow the project's coding style and conventions.

### 5. Test Your Changes
   - Ensure all existing and new tests pass by running:
     
     ```bash
     npm test
     ```

### 6. Commit Your Changes
   - Commit your changes with a clear and descriptive commit message:
     
     ```bash
     git commit -m "Add some feature"
     ```

### 7. Push to Your Fork
   - Push your changes back to your forked repository:
     
     ```bash
     git push origin feature/your-feature-name
     ```

### 8. Open a Pull Request
   - Go to the original repository (ambr9n/overwatch-tac) and click on the **Pull Requests** tab. Then, click the **New Pull Request** button. 
   - Select your branch and describe your changes in the pull request template.

## Pull Request Guidelines
- Ensure your pull request is focused on a single change or feature.
- Write a clear description of the changes and why they are needed.
- If applicable, link to any relevant issues.

## Code of Conduct
Please make sure to read our [Code of Conduct](link-to-code-of-conduct) before contributing. We expect all participants to be respectful and inclusive.

## Contact
For any questions or concerns, feel free to reach out to the project maintainers or create an issue in the repository. 

Thank you for contributing!
