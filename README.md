# Overwatch Tac

A tactical analysis and visualization tool for Overwatch built with TypeScript, designed to provide comprehensive stat tracking and advanced game insights.

## 🚀 Features

- **Advanced Analytics** - Detailed statistical analysis of gameplay and performance metrics
- **Real-time Visualization** - Interactive charts and graphs for game data
- **TypeScript-Powered** - Type-safe codebase for reliability and maintainability
- **Responsive Design** - Optimized for desktop and mobile viewing
- **Tactical Insights** - Strategic recommendations based on collected data

## 💻 Tech Stack

- **TypeScript** (94.3%) - Core application logic
- **CSS** (4.1%) - Styling and design
- **JavaScript** (1.3%) - Additional scripting
- **React** - UI framework
- **Node.js** - Runtime environment

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/ambr9n/overwatch-tac.git
cd overwatch-tac
```

Install dependencies:
```
npm install
```

Start dev server:
```
npm start
```

The application will be available at ``http://localhost:3000``
